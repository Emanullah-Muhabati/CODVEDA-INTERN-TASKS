# Simple Static Web Project

A small front-end project with a single-page site using HTML, CSS, and JavaScript.

## Description

This repository contains a **responsive landing page** created as part of the **Codveda Internship Program**.  

The project demonstrates front‑end development skills, responsive design techniques, and clean documentation practices.

This repository contains a minimal static website used for learning and small demos. It includes a basic layout, styles, and a script for interactivity.

![Desktop Screenshot](./Screenshot%202026-07-13%20084611.png)

## 🚀 Features

- Fully responsive design (mobile, tablet, desktop)
- Modern layout using HTML5, CSS3, and JavaScript
- Optimized for performance and accessibility
- Easy to customize and extend for future projects

## Files

- [index.html](index.html) — main HTML page
- [styles.css](styles.css) — stylesheet for layout and appearance
- [script.js](script.js) — JavaScript for page behavior

## Usage

1. Open [index.html](index.html) in your web browser (double-click or use "Open File").
2. Edit `styles.css` and `script.js` to modify the look and behavior.

## Development

- This is a static site — no build step required.
- Use any editor to change files and refresh the browser to see updates.
